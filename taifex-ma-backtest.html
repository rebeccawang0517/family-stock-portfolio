「現在／近N根」欄✔＝判讀範圍內（看最近 N 根）有出現過該訊號；範圍設大時幾乎都會打勾，想看「當下這一根」請把「看最近」設 1。組合統計＝過去每次「勾選條件同時成立」的那根K棒，往後 N 根收盤的漲跌；樣本 ≥30 才較可信。這是訊號組合本身的邊際優勢，跟你設的停損停利無關。

🩺 常見排列勝率體檢（哪些一定做、哪些別碰）

「方向勝率」＝這個排列出現後 N 根，往它預期方向（多→漲、空→跌）的機率；「頻率」＝多久出現一次（對照你「每天 1~3 筆」）。樣本 ≥30 才可信。這是進場後的方向優勢（不含你的移動停損鎖利，含鎖利的真實勝率去②跑那組）。清單下方的逆勢／亂做例子會自然變 ❌＝「別碰」。

🧭 操作流程（不知道先做什麼就照這個跑；點標題可收合）
你的目標＝高勝率低頻率。路線：先找勝率高的排列 →（可選）電腦再優化 → 手動設好進出場＋再次進場＋移動停損並驗證 → 算倉位 → 看盤輔助。手動／自動／再進／倉位的分工看最下面。
0
匯入資料　各週期K棒進來（日／60／30／15／5分）。
1
找高勝率排列　一鍵體檢常見排列，看哪些 ✅一定做、哪些 ❌別碰，挑一組（可直接「↪套用最佳那組」帶回②）。
2
（可選）電腦自動找更好　大量搜組合，在樣本內找最佳、樣本外驗證，挑樣本外也穩的，按「套用」帶回②。
3
手動設定＋驗證（②手動回測頁）　精靈「1 進場」「2 出場」填好條件、勾移動停損鎖利 → 按 ▶ 執行回測，看勝率／回撤／逐筆。這步就是「手動回測」。
4
算倉位（②精靈第 3 步）　回測一跑完，倉位試算自動用這組 edge 算首單／加碼／口數／風險紅線與可行性。
5
看盤下單輔助　當下這根符不符合、歷史類比、壓撐位置。
名詞分工（釐清跳來跳去）：
• 自動回測（③）＝電腦大量搜「探索」找候選，怕過度最佳化，一定看樣本外。
• 手動回測（②）＝把選定的一組「驗證＋微調」，看完整逐筆；跑完才能算倉位。
• 再次進場＝不是獨立步驟，是進場設定的一部分（②精靈「1 進場」下方那塊）：空手符合【首次進場】才進 → 持倉只看出場 → 出場後要符合【再次進場】才重進（沒勾就沿用首次，用意是避免被洗掉又立刻追）。
• 倉位（第4步）＝一定要先有回測結果才算得出，所以永遠放最後。

🟪 進出場條件（②手動回測勾選框的定義）

MA1~MA5 ＝ 你在「均線參數」設的值（預設 MA1=5、MA2=10、MA3=20、MA4=60、MA5=240長期線）。標籤上的「MA20/60/240」是用預設值命名，改了均線參數請以 MA3/MA4/MA5 為準。

條件定義（怎麼算才算成立）常用在
多排均線多頭排列：MA1>MA2>MA3>MA4（「多空排用幾條」設3條時只看 MA1>MA2>MA3）做多進場
空排均線空頭排列：MA1<MA2<MA3<MA4（設3條時只看 MA1<MA2<MA3）做空進場／做多出場
多排消失 / 空排消失原本成立的多排／空排，這根不再成立出場
紅K / 黑K紅K＝收盤>開盤；黑K＝收盤<開盤進出場
強紅K（大陽線）紅K，且實體(收−開) ≥ 整根K棒高低幅的 60%（力道強、少上下影）做多進場
強黑K（大陰線）黑K，且實體 ≥ 高低幅的 60%做空進場
長下影下影線 ≥ 整根K棒高低幅的 50%（下方一度殺低又被買回＝探底/撐住訊號）做多進場
長上影上影線 ≥ 高低幅的 50%（上方一度衝高又被壓回＝賣壓/見頂訊號）做空進場
站上 MA5 / MA10 / MA20 / MA60 / MA240收盤 > MA1(5) / MA2(10) / MA3(20) / MA4(60) / MA5(240)做多進場／做空出場（短均線＝更靈敏）
跌破 MA5 / MA10 / MA20 / MA60 / MA240收盤 < MA1(5) / MA2(10) / MA3(20) / MA4(60) / MA5(240)做多出場／做空進場
黃金交叉 (MA1↑MA2)MA1 由下往上穿過 MA2（快線升破慢線）做多進場／做空出場
死亡交叉 (MA1↓MA2)MA1 由上往下跌破 MA2（快線跌破慢線）做多出場／做空進場
MA支撐盤中最低觸及 MA20 或 MA60 附近（容差內），但收盤仍站在該均線之上（被均線撐住沒跌破）做多進場
MA壓力盤中最高觸及 MA20 或 MA60 附近，但收盤仍在該均線之下（被均線壓回沒站上）做空進場
走勢向上 ↑ / 向下 ↓最近 N 根收盤的線性回歸斜率（N＝「走勢判斷回看」）：斜率 > 走勢門檻＝向上、< −門檻＝向下。門檻＝每根平均要漲/跌超過幾點才算，0＝只要斜率不為0（盤整易翻可調大）順勢過濾
放量這根成交量 ≥ 前20根均量的 1.5 倍配紅K/多排＝有量的攻擊
爆量這根成交量 ≥ 前20根均量的 2.5 倍（量能大增）配紅K偏續攻、配黑K偏轉弱
連續2/3紅K・連續2/3黑K這根與前 1（或前 2）根都同方向收紅／收黑（動能延續）追多／追空
貼近期高（突破）收盤落在「近20根最高價（含這根）」的 0.5% 內＝收在近期高檔。含這根是刻意設計：這根創新高且收高檔＝突破確認、沒創新高＝逼近前高。★用的是已收完K棒的高點（收盤時就已知），不是未來函數——跟紅K/強紅K/長上影同一個因果標準（都用當根已收完的OHLC），成交也在收盤或次根開盤之後，不會偷看未來做多進場（突破追價）
貼近期低（破底）收盤落在「近20根最低價（含這根）」的 0.5% 內＝收在近期低檔＝破底或逼近前低。同樣用已收完K棒的低點，因果安全做空進場（破底追空）

名詞意思怎麼算注意
軌跡長度拿最近幾根當「現在這段」1天/3天/1週 會用「這週期每天幾根」的中位數換算成根數，也可自訂根數太短(如日線1天=1根)沒形狀可比，會自動至少3根
